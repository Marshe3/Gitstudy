## 🚀 주요 프로젝트 (Projects)

### ⚔️ Project: Shadow Blade (3D Action RPG)
* **개발 기간**: 202X.0X ~ 202X.0X (X개월)
* **팀 구성**: 4인 프로젝트 (메인 클라이언트 프로그래머 담당)
* **사용 기술**: Unreal Engine 5, C++, Blueprint

**[주요 역할 및 성과]**
* **GAS(Gameplay Ability System) 도입**: 캐릭터의 스킬, 버프/디버프 상태 이상 시스템을 모듈화하여 개발 속도를 높이고 코드의 재사용성을 확보했습니다.
* **AI Behavior Tree 구현**: 보스 몬스터의 패턴 및 위협 수준(Aggro) 시스템을 설계하여 역동적인 전투 환경을 구성했습니다.
* **최적화**: Unreal Insights를 활용하여 병목 구간을 파악하고, Object Pooling을 적용해 평균 프레임을 안정화했습니다.

### 🌐 Project: Arena Brawl (Multiplayer Brawler)
* **개발 기간**: 202X.0X ~ 202X.0X (X개월)
* **팀 구성**: 개인 프로젝트
* **사용 기술**: Unreal Engine 4, C++, Dedicated Server

**[주요 역할 및 성과]**
* **네트워크 동기화**: 리플리케이션(Replication)과 RPC를 활용하여 클라이언트 간 캐릭터 이동 및 피격 판정을 동기화했습니다.
* **UI/UX 구현**: UMG를 활용하여 가변 해상도에 대응하는 반응형 인터페이스를 구축했습니다.
